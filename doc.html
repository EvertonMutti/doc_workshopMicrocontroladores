Introdução

O objetivo deste projeto é criar um varal de roupas automático que possa ser controlado por um botão para retrair e expandir, mas que tenha também um modo automático para detectar quando está chovendo e retrair o varal automaticamente, e vice versa a partir de microcontrolador Arduino.

Requisitos Funcionais

  • O varal deve possuir modo automático e manual;
  • O varal deve ser capaz de se expandir e retrair por meio de botôes;
  • O varal deve possuir um sensor de água para detectar quando está chovendo;
  • O microcontrolador Arduino deve ser capaz de receber informações do sensor de água e acionar o motor do varal para retrair ou expandir se estiver no modo automático;

Requisitos Não Funcionais

  • O sistema deve ser seguro para os usuários e não apresentar riscos de choques elétricos ou outros tipos de acidentes.

    obs: enrola todos os cabos em Fita adesiva de vinil e colocar o arduino em um local que não tome chuva

Funcionalidades

  • Os botões devem permitir que o usuário controle o varal manualmente;
  • O sensor de água deve enviar informações para o microcontrolador, que acionará o motor do varal para retrair ou expandir conforme necessário e se estiver no modo automático;

Tecnologias e Componentes Utilizados

  • Motor DC para o acionamento do varal;
  • Sensor ultrassônico para medir a distância do varal;
  • Interruptor;
  • Ponte H;
  • Led;
  • Resistors;
  • Sensor de umidade para detectar a presença de chuva;
  • Microcontrolador Arduino para processar as informações do sensor e controlar o motor do varal;
  • Materiais para construir o varal, como cabos de aço e suportes;
  • Fita adesiva de vinil
  • Prontoboard.
